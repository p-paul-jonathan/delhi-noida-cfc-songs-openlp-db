# Delhi - Noida CFC songs openlp database

## Installation instructions

- Open OpenLP ![](./openlp.png)
- Go to Tools > Open Data Folder
- In the folder open the `songs` folder
- save the existing `songs.sqlite` file as a backup (something like `songs.sqlite.bak`)
- copy the `songs.sqlite` in this repository to the data/songs folder
- Restart OpenLP

## Contribution instructions

- Go to Tools > Open Data Folder
- Open the `songs` folder
- copy the `songs.sqlite`
- overwrite the `songs.sqlite` file in this folder with the file in the OpenLP Data folder
- Go to OpenLP
- Go to Tools > Songs > Song List Report
- It will open a dialoge to save `song_extrac.csv`
- save it in this directory
- open a Pull Request
